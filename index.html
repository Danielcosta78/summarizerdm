API Integration

Use SummarizerDM in your projects with this simple configuration:

<script>
  src="https://cdn.jsdelivr.net/gh/Danielcosta78/summarizerdm@main/cdn/summarizerdm.js"
</script>
<script>
  Summarizer.initialize({
    inputId: 'textInput',
    outputId: 'summaryOutput',
    sliderId: 'summaryLevel',
    buttonId: 'summarizeBtn',
    licenseKey: 'REMOVE-WATERMARK'
  });
</script>

Configuration Options:

inputId: ID of the textarea element for input text
outputId: ID of the div element where summary will appear
sliderId: ID of the range input for summary length
buttonId: ID of the button to trigger summarization
licenseKey: Add to 'LICENSEKEY' to remove watermark
